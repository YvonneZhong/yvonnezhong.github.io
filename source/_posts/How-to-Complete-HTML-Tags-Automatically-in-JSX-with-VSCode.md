1. In VSCode, go to `Settings/Extensions/Emmet`
2. Click `Edit in settings.json`
3. Add below code into `settings.json`
   ```json
    "emmet.includeLanguages": {
    "javascript": "javascriptreact"
   }
   ```
4. Restart VSCode.
